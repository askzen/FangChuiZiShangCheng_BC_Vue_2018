项目介绍:  
    使用技术: vue2.0CLI + vueRouter + axios + sass.  
    实现功能: 商品首页展示,商品列表展示,购买详情展示.  
第一步运行npm install下载依赖包.  
第二步 运行npm run serve 启动项目.  
适合vue初学者练习的项目(尤其页面结构布局).
